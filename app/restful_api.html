RESTful API

A robust RESTful API built with PHP, designed to power mobile and web applications. It features secure authentication, data validation, and scalable endpoints for various resources.

Key Features

  • JWT-based authentication
  • CRUD operations for multiple resources
  • Rate limiting and API key management
  • Comprehensive error handling
  • OpenAPI (Swagger) documentation
  • Unit and integration tests

Technologies Used

  • PHP (Slim Framework)
  • MySQL
  • Tailwind CSS (for documentation site)
  • JavaScript (for interactive docs)
  • Docker
