Content Management System

A custom CMS built with PHP, designed for easy content editing, user role management, and media uploads. The system is secure, extensible, and optimized for fast publishing workflows.

Key Features

  • Rich text editor for content creation
  • Role-based access control
  • Media library for images and files
  • SEO tools and metadata management
  • Customizable templates and themes
  • Revision history and content scheduling

Technologies Used

  • PHP (Symfony Framework)
  • MySQL
  • Tailwind CSS
  • JavaScript (Alpine.js)
  • Docker
